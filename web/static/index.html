Wie das hier funktioniert

Kurz erklärt, ganz ohne Mathe-Studium. Von „warum macht man sowas" bis „woher weiß man, dass die Ziffern wirklich stimmen".

01

Warum π endlos rechnen?

Ehrlich gesagt: weil man's kann. Und weil π ein richtig guter Dauertest für Rechner ist. Die Ziffern hören nie auf und wiederholen sich nie. Eine Aufgabe, die nie langweilig wird und nie fertig ist.

02

Der Spigot-Trick

Statt riesige Zahlen im Speicher zu halten, spuckt ein Spigot-Algorithmus die Ziffern einzeln aus und wirft jede alte gleich wieder weg. Dadurch braucht er fast keinen Arbeitsspeicher und läuft sogar auf einem kleinen Raspberry Pi.

03

Sofort gegengeprüft

Jeder fertige Block von 1000 Stellen wird live mit der offiziellen Datenbank von pi.delivery (Google) abgeglichen. Stimmt etwas nicht, steht es sofort im Status. So weißt du: Was hier steht, ist echtes π.

04

Immer im Kreis

Damit der kleine Server nicht an Billionen Stellen erstickt, hört er nach einer festen Zahl auf, springt zurück auf die erste Stelle und legt wieder los. Ein Zähler merkt sich, in welcher Runde er gerade ist.

05

Wo das Ganze läuft

Auf einem ganz normalen Heimserver, erreichbar über die lokale IP und nach außen sauber abgesichert durch einen Cloudflare-Tunnel. Kein Rechenzentrum, kein großes Gerät, einfach eine Kiste, die durchläuft.

06

Dein Benchmark

Dein Browser rechnet 15.000 Stellen direkt auf deinem Gerät und stoppt die Zeit auf die Millisekunde. Das misst die rohe Single-Core-Leistung deines Prozessors. Je schneller, desto stärker das Gerät. Gerechnet wird bei dir, nie am Server.

Benchmark

Wie schnell ist dein Rechner?

Dein Browser rechnet 15.000 Stellen von π aus, direkt auf deinem Gerät, und stoppt die Zeit auf die Millisekunde. Trag einen Namen ein und schau, wo du gegen die anderen landest.

Im Hintergrund macht das die Mathe-Engine deines Browsers (V8 bei Chrome, JavaScriptCore bei Safari). Gemessen wird damit die reine Single-Core-Rohleistung deines Prozessors. Je schneller die Zeit, desto stärker das Gerät.

Der π Random Walk

Eine schöne Art, π anzuschauen: Jede Ziffer (0–9) zeigt in eine andere Richtung im Raum. Hängt man die Schritte aneinander, entsteht ein einzigartiger Pfad durch die Stellen von π. Ziehen zum Drehen, scrollen zum Zoomen.
